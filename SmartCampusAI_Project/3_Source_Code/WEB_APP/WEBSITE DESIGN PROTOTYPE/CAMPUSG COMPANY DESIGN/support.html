Popular Troubleshooting

I cannot log in to the administrator dashboard.
Please ensure you are using the correct URL specific to your institution. If the issue persists, try clearing your browser cache and cookies, then attempt to reset your password using the "Forgot Password" link on the login screen.
How do I schedule a new course semester?
Navigate to the **Academics** module, select **Timetabling**, and click "New Semester Setup." Our system will guide you through resource allocation, faculty assignment, and course mapping. Refer to the 'Course Scheduling Guide' in the Knowledge Base for detailed steps.
Where can I download the latest API documentation?
The full REST API documentation, including endpoints and authentication details, is available in the **Developers Section** of the Knowledge Base. You must have Premium or Enterprise access rights to view and utilize the API.
